# eON-INDOOR_CLIMATE_SENSOR_EFEKTA_eON29z
It's a fully assembled and programmed compact wireless devices, atmospheric pressure, temperature, humidity, weather forecaster, illumination, charts with 2.9" e-paper display, resolution of 296 x 128 pixels. The device has an industrial look, at least I wanted to achieve this.  The sensor has good energy efficiency. Powered by CC2530 chip (Zigbee).
